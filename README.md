# P2P File Sharing Chat (C)

TCP 기반 로그인 채팅 시스템에 **P2P 파일 전송 기능**을 결합한 C 네트워크 프로젝트입니다.  
로그인 후 하나의 채팅방에서 대화할 수 있으며, 특정 사용자에게 `.txt` 파일을 직접 전송할 수 있습니다.

---

## 주요 기능

- **로그인 시스템**
  - 서버 기반 ID / PW 인증
- **단일 채팅방**
  - 접속한 모든 클라이언트에게 메시지 브로드캐스트
- **P2P 파일 전송**
  - 서버는 피어 정보(IP/PORT)만 중계
  - 클라이언트 간 직접 연결로 `.txt` 파일 공유

---

## 동작 구조

- **채팅**
  - TCP 기반 서버–클라이언트 구조
  - `select()`를 이용한 다중 클라이언트 처리
- **P2P 파일 전송**
  - 클라이언트는 로그인 후 P2P 리스너 실행
  - 파일 요청 시 서버가 상대에게 피어 정보 전달
  - 클라이언트 간 직접 TCP 연결
